# Contributing to ELMS

Thank you for contributing to the ELMS project.

## How to contribute

1. Fork the repository on GitHub.
2. Clone your fork locally.
3. Create a new branch with a descriptive name, for example `contribute-4`.
4. Make your changes in a focused, small commit.
5. Push the branch to GitHub.
6. Open a pull request against the `master` branch.

## Local setup

1. Open the project folder in VS Code.
2. Run the task: `Tasks: Run Task` → `Start Python HTTP server`.
3. Open `http://127.0.0.1:8000/` in your browser.

## Code style

- Keep HTML semantic and accessible.
- Keep CSS class names meaningful and consistent.
- Keep JavaScript modular and readable.

## Commit message format

Use clear, concise commit messages. Examples:

- `docs: add contributing guide`
- `chore: update page metadata`
- `fix: correct demo credentials`
- `style: improve button spacing`

## Pull request

Please include a short description of your change and any testing steps.
