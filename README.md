# ELMS — E-Learning Management System

This is a static demo project for an E-Learning Management System built with vanilla HTML, CSS, and JavaScript.

## Project structure

- `index.html` — main entry point
- `css/styles.css` — app styling
- `js/app.js` — application logic, data, and UI rendering
- `.vscode/launch.json` — VS Code debug configuration
- `.vscode/tasks.json` — local server task
- `vercel.json` — Vercel deployment configuration

## How to run locally

1. Open the project folder in VS Code.
2. Run the task: `Tasks: Run Task` → `Start Python HTTP server`.
3. Open `http://127.0.0.1:8000/` in your browser.

## Vercel deployment

The project is configured for Vercel as a static site. After pushing to GitHub:

1. Import the repository into Vercel.
2. Use the default static site deployment.
3. The project will serve `index.html` at the root.
